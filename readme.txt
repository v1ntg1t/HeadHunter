1. MinDistance
	Problem: find min distance between couples of points on the plane
	Solution: Preparata's algorithm, http://e-maxx.ru/algo/nearest_points
	Input data file: res/in.txt
	Command scripts:
		clean - remove binary classes
		gen N - generate input data file with N points (N - argument of command)
		make - build binary classes
		run - run binary classes

2. Fraction
	Problem: translate fraction into given number system with periodic brackets
	Solution: division in a column using given radix
	Input data file: in.txt
	Command scripts:
		clean - remove binary classes
		make - build binary classes
		run - run binary classes
